## Welcome to Cifera Shop!

Cifera Shop is a landing page of one of the biggest online shops in Southeast Asia!

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How To Run Unit Test

Run this:

```bash
npm test
# or
yarn test
# or
pnpm test
# or
bun test
```

## Code Explanation

This repository is written using Next.js, utilize the server side rendering in order to improve FCP and SEO, so it will fulfill its purpose as an online shop!

The products and images are rendered using server components, so the SEO score will be high and the products and images can be retrieved in Search Engines, and the interactivity is put inside the client component to utilize react state to open modal
